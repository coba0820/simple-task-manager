# シンプルタスク管理ツール

ブラウザだけで動く、非エンジニア向けのシンプルなタスク管理アプリです。
`index.html` 1ファイルだけで完結しており、サーバーやインストール不要ですぐに使えます。

---

## 1. 完成している機能

### 基本機能
- ✅ タスクの追加(入力欄に文字を入れて「追加」ボタン or Enterキー)
- ✅ タスク一覧の表示
- ✅ チェックボックスで完了・未完了の切り替え
- ✅ タスクの削除(🗑 削除ボタン、削除前に確認ダイアログが出ます)
- ✅ **localStorage** に自動保存 → ブラウザを閉じても、再度開いた時にタスクが残る
- ✅ シンプルで見やすいデザイン
- ✅ スマホ・タブレット・PCに対応したレスポンシブデザイン

### 今回追加した機能
- ✅ **タスクの編集機能**:各タスクの「✏️ 編集」ボタンを押すと、その場でタスク名・期限・優先度を書き換えられます(インライン編集)。「保存」で確定、「キャンセル」で編集を取り消せます。
- ✅ **期限(締切日)の設定**:タスク追加時・編集時にカレンダーから日付を選択できます(任意入力)。期限が過ぎていて未完了のタスクは、カードの左に赤いラインと「⚠ 期限切れ」バッジが表示されます。
- ✅ **優先度の設定**:「高・中・低」の3段階から選択できます。各タスクに色分けされたバッジ(高=赤、中=オレンジ、低=青)が表示されます。
- ✅ **検索機能**:検索ボックスにキーワードを入力すると、タスク名に部分一致するものだけがリアルタイムで絞り込まれます。
- ✅ **並び替え機能**(おまけで追加):「追加順」「期限が近い順」「優先度順」から選んで並び替えができます。
- ✅ 件数表示に「⚠ 期限切れ○件」を追加表示
- ✅ **表示フィルターの記憶**:「すべて／未完了／完了済み」のどれを選んでいたかを記憶し、ブラウザを閉じて次回開いたときも同じ表示状態から始まります。
- ✅ **編集・削除ボタンをアイコンのみに変更**:文字ラベルを無くし、✏️(編集)・🗑(削除)のアイコンのみのボタンにして、サイズも少しコンパクトにしました。

## 2. 使い方

1. `index.html` をダブルクリックして、ブラウザ(Chrome、Safariなど)で開く
2. タスク名を入力欄に入力
3. 必要であれば「期限」でカレンダーから日付を選択(空欄でもOK)
4. 「優先度」で高・中・低を選択(初期値は「中」)
5. 「追加」ボタンを押す(またはタスク名入力後にEnterキー)
6. タスクが完了したら、左側のチェックボックスをクリック → 完了扱い(取り消し線がつく)
7. タスクを直したい場合は「✏️ 編集」ボタンを押す → 入力欄が表示されるので書き換えて「保存」
8. 不要になったタスクは「🗑 削除」ボタン → 確認ダイアログで「OK」を押すと削除
9. 検索ボックスにキーワードを入れると、一致するタスクだけに絞り込まれる
10. 「すべて／未完了／完了済み」ボタンで表示を絞り込み、右側の並び替えメニューで表示順を変更できる
11. ブラウザを閉じても、次回同じブラウザで `index.html` を開けばタスクはそのまま残っています

### 注意点
- タスクの保存はブラウザの **localStorage** という仕組みを使っています。これは「今使っているブラウザ・今使っている端末」にのみ保存されます。
  - 別のパソコンやスマホから開くと、タスクは共有されません(それぞれの端末・ブラウザごとに別々に保存されます)。
  - ブラウザの「閲覧履歴やキャッシュの削除」を行うと、保存したタスクも消える場合があります。
- 以前のバージョンで保存したタスク(期限・優先度が無いデータ)を開いても、自動的に優先度「中」・期限なしとして扱われるので、データが消えることはありません。

## 3. コードを自分でカスタマイズしたい場合

`index.html` の中に日本語コメントを入れてあります。主に以下の箇所を編集すれば見た目や初期設定を変更できます。

### 色を変えたいとき
`<style>` タグの中、以下のコメントで囲まれた部分にある色コード(`#4f46e5` など)を書き換えるだけです。優先度バッジの色もここで変更できます。

```
/* ▼▼▼ ここから「見た目(デザイン)」の設定エリアです ▼▼▼ */
:root {
  --color-primary: #4f46e5;         ← メインカラー
  --color-priority-high: #ef4444;   ← 優先度「高」バッジの色
  --color-priority-medium: #f59e0b; ← 優先度「中」バッジの色
  --color-priority-low: #3b82f6;    ← 優先度「低」バッジの色
  ...
}
/* ▲▲▲ デザイン設定エリアはここまで ▲▲▲ */
```

### タイトルや説明文を変えたいとき
`<body>` 内の以下の部分を書き換えます。

```html
<header class="app-header">
  <h1>📝 シンプルタスク管理</h1>   ← タイトルを変更できます
  <p>やることをサクッと管理しよう</p> ← 説明文を変更できます
</header>
```

### 優先度の初期値を変えたいとき
追加フォームの `<select id="task-priority-input">` にある `selected` を、別の `<option>` に付け替えると初期選択が変わります。

### 機能(動き)の部分
`<script>` タグの中、以下のコメントで囲まれた部分に機能のコードがあります。基本的に編集不要ですが、関数ごとにコメントを入れているので、処理内容を確認できます。

```
/* ▼▼▼ ここから「機能(動き)」の設定エリアです ▼▼▼ */
...
/* ▲▲▲ 機能設定エリアはここまで ▲▲▲ */
```

## 5. 技術情報

| 項目 | 内容 |
|---|---|
| 構成ファイル | `index.html`(HTML / CSS / JavaScript すべて1ファイルに内包) |
| データ保存先 | ブラウザの `localStorage`(キー名: `simple-task-manager-tasks`) |
| データ構造 | `{ id: string, text: string, completed: boolean, dueDate: string, priority: "high"\|"medium"\|"low" }` の配列 |
| 表示状態の保存 | 選択中の表示フィルター(すべて／未完了／完了済み)を `localStorage`(キー名: `simple-task-manager-filter`)に保存 |
| 外部ライブラリ | 使用なし(プレーンなHTML/CSS/JavaScriptのみ) |
| 対応環境 | モダンブラウザ(Chrome, Safari, Edge, Firefoxなど)、PC・スマホ・タブレット対応のレスポンシブデザイン |

## 6. 公開(デプロイ)について

このアプリをインターネット上に公開して、URLで誰でもアクセスできるようにしたい場合は、
**Publishタブ** から公開してください。ワンクリックで公開処理が行われ、公開後のURLが発行されます。
